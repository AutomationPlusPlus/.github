# automation++ ¯\\\_(ツ)\_/¯

Automation with extra plusses.

We write tools that do the boring stuff so we don't have to — infrastructure
automation, config wrangling, and the occasional TUI that absolutely did not
need to be that fancy.

**Why "++"?** Because plain automation wasn't enough, and `automation--` sends
the wrong message.
